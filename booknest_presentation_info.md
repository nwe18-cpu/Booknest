# Booknest Project Presentation Key Points

ဒီအချက်အလက်တွေကို Presentation AI (ဥပမာ - Gamma, Tome စသည်) ထဲသို့ ထည့်သွင်းပြီး PowerPoint Slide များ ပြုလုပ်ရန် အသုံးပြုနိုင်ပါသည်။

---

1. **Backend နည်းပညာနှင့် Database (Backend Technology & Database)**
   - Backend ကို ခေတ်မီပြီး လုံခြုံစိတ်ချရသော **Laravel 11 (PHP 8.2)** Framework ဖြင့် ရေးသားထားပြီး ဒေတာဘေ့စ်အတွက် **MySQL** ကို အသုံးပြုထားပါသည်။

2. **စနစ်သုံးစွဲသူ ကဏ္ဍနှစ်ခု (Dual-Portal System)**
   - ဝယ်ယူသူများ စာအုပ်ရှာဖွေဝယ်ယူရန်အတွက် Customer Portal နှင့် စာအုပ်ဆိုင်လုပ်ငန်းတစ်ခုလုံးကို ထိန်းချုပ်စီမံရန်အတွက် Staff/Admin Panel ဟူ၍ သီးခြားစီမံခန့်ခွဲမှုစနစ် နှစ်ခု ပါဝင်ပါသည်။

3. **ဈေးဝယ်လှည်းနှင့် အော်ဒါစနစ် (E-Commerce & Cart Management)**
   - စာအုပ်များကို ဈေးဝယ်လှည်း (Cart) ထဲသို့ ထည့်ခြင်း၊ အရေအတွက် ပြင်ဆင်ခြင်း၊ ပယ်ဖျက်ခြင်းများအပြင်၊ ပို့ဆောင်မည့်လိပ်စာ (Shipping Address) များကို သိမ်းဆည်းထားနိုင်ခြင်း (Save Address)၊ ပြင်ဆင်ခြင်း၊ ဖျက်ခြင်းနှင့် Default Address သတ်မှတ်ခြင်းများ ပြုလုပ်နိုင်သော လိပ်စာစီမံခန့်ခွဲမှုစနစ် ပါဝင်ပါသည်။

4. **ငွေပေးချေမှုစနစ် အမျိုးမျိုး (Diverse Payment Options)**
   - သုံးစွဲသူများ အဆင်ပြေချောမွေ့စွာ ငွေချေနိုင်ရန်အတွက် Online Payment Gateway ဖြစ်သော **Stripe**၊ ပြည်တွင်းမိုဘိုင်းလ်ငွေပေးချေမှုများဖြစ်သည့် **KBZPay (Kpay)** နှင့် **WavePay** တို့အပြင် ပစ္စည်းရောက်မှ ငွေချေစနစ်ဖြစ်သော **Cash on Delivery (COD)** အပါအဝင် စုစုပေါင်း ငွေပေးချေမှုပုံစံ ၄ မျိုးဖြင့် စိတ်ကြိုက် ရွေးချယ်ပေးချေနိုင်ပါသည်။

5. **အွန်လိုင်း PDF ဖတ်ရှုခြင်းနှင့် ဒေါင်းလုဒ်စနစ် (Interactive PDF Reader & Downloads)**
   - ဝယ်ယူထားသော စာအုပ်များကို Browser ထဲတွင် တိုက်ရိုက်ဖတ်ရှုနိုင်ရန် PDF Streaming စနစ် ပါဝင်သလို၊ Offline ဖတ်ရှုရန်အတွက် ဒေါင်းလုဒ် (Download) ဆွဲနိုင်သော စနစ်လည်း ပါဝင်ပါသည်။

6. **စာဖတ်နှုန်းတိုးတက်မှု မှတ်တမ်းတင်စနစ် (Reading Progress & Bookmarks)**
   - Customer တစ်ဦးချင်းစီ စာအုပ်ဖတ်နေသည့် စာမျက်နှာ (Current Page) ကို မှတ်သားပေးခြင်း၊ စာညှပ် (Bookmark) ပြုလုပ်နိုင်ခြင်းနှင့် စာအုပ်ဖတ်ပြီးမြောက်မှု (Reading Progress) ကို ခြေရာခံနိုင်ခြင်း။

7. **VIP အဖွဲ့ဝင်စနစ် (Premium Subscription System)**
   - အခပေး Membership အစီအစဉ်များ (Subscription Plans) ကို ဝယ်ယူနိုင်ပြီး သတ်မှတ်ထားသော သက်တမ်းအလိုက် စာအုပ်များကို အကန့်အသတ်မရှိ ဖတ်ရှုနိုင်မည့် VIP စနစ် ပါဝင်ပါသည်။

8. **စာအုပ်နှင့် ကဏ္ဍခွဲများ စီမံခန့်ခွဲခြင်း (Catalog & Classification Management)**
   - Admin များအနေဖြင့် စာအုပ်များ (Items)၊ စာရေးဆရာများ (Authors)၊ Promotion Banner များနှင့် စာအုပ်အမျိုးအစားများ (Classifications) ကို လွယ်ကူစွာ စီမံခန့်ခွဲနိုင်ပါသည်။

9. **သုံးသပ်ချက်နှင့် ထိန်းညှိမှုစနစ် (Reviews & Moderation)**
   - Thုံးစွဲသူများက စာအုပ်များအပေါ် ကြယ်ပွင့်ပေးခြင်း (Rating) နှင့် ထင်မြင်ချက်ပေးခြင်း (Reviews) များ ပြုလုပ်နိုင်ပြီး Admin က မเหมาะสมသော Review များကို ပြန်လည်စစ်ဆေး ဖျက်သိမ်းနိုင်ပါသည်။

10. **လုပ်ဆောင်ချက်မှတ်တမ်းနှင့် အသိပေးချက်များ (Activity Log & Notifications)**
    - Admin/Staff များ၏ လုပ်ဆောင်ချက်များကို ခြေရာခံနိုင်သည့် Staff Activity Log စနစ်နှင့် သုံးစွဲသူများထံ အရေးကြီးအချက်အလက်များ ပေးပို့ရန် Notification စနစ်တို့ ပါဝင်ပါသည်။

11. **ဝန်ထမ်းလုပ်ပိုင်ခွင့် ခွဲခြားသတ်မှတ်မှုစနစ် (Admin & Staff Role Authorization)**
    - Admin နှင့် Staff တို့၏ လုပ်ပိုင်ခွင့်ကို တိကျစွာ ခွဲခြားသတ်မှတ်ထားသည်။ မှာယူမှုများ (Order Management)၊ VIP customer စာရင်းများ၊ သုံးစွဲသူများစီမံခန့်ခွဲမှု (Customer Management) နှင့် ဝန်ထမ်းအကောင့်များစီမံခန့်ခွဲမှု (Staff Management) တို့ကို Admin သီးသန့်သာ လုပ်ဆောင်ခွင့်ပြုထားသည်။

12. **နောင်ထပ်တိုးချဲ့မည့် အစီအစဉ်များ (Future Scope & Roadmap)**
    - **ပြည်တွင်း Mobile Wallet များ တိုက်ရိုက်ချိတ်ဆက်ခြင်း (Kpay & WavePay API Integration):** လက်ရှိ simulation အဆင့်မှသည် တကယ့် API Gateway များနှင့် ချိတ်ဆက်ပြီး online မှ တိုက်ရိုက် ငွေပေးချေနိုင်စေရန်။
    - **အော်ဒီယိုစာအုပ်စနစ် (Audiobooks Support):** E-book များအပြင် စာနားဆင်လိုသူများအတွက် Audiobooks များကို နားထောင်နိုင်မည့် Audio Player စနစ် ထည့်သွင်းရန်။
    - **AI စာအုပ်အကြံပြုစနစ် (AI-Based Recommendation Engine):** အသုံးပြုသူ ဖတ်ဖူးသမျှ စာအုပ်အမျိုးအစားများနှင့် Ratings များအပေါ် အခြေခံပြီး စာအုပ်အသစ်များကို AI ဖြင့် အလိုအလျောက် ညွှန်းဆိုပေးရန်။
    - **အရောင်းနှင့် စာရင်းအင်းပြဇယားများ (Advanced Analytics & Charts):** Admin Panel တွင် လစဉ်ရောင်းအားနှင့် VIP subscription အခြေအနေများကို စာရင်းဇယားပြဇယားများ (Visual Charts) ဖြင့် ကြည့်ရှုနိုင်ရန်။
    - **အင်တာနက်မရှိဘဲ ဖတ်ရှုနိုင်သောစနစ် (PWA & Offline Reading):** PWA (Progressive Web App) နည်းပညာ သုံးပြီး ဒေါင်းလုဒ်လုပ်ထားသော စာအုပ်များကို အင်တာနက်မရှိချိန် (Offline) တွင်ပါ ဖတ်ရှုနိုင်စေရန်။
